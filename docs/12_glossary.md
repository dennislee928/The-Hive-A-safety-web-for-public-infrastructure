# Glossary

- **CAP**: Common Alerting Protocol
- **Route 1**: Baseline public warning channels without app requirement
- **Route 2**: Optional citizen app for bidirectional interaction
- **Dead-man keepalive**: hold-to-maintain elevated state; loss triggers rollback
- **Dual control**: two authorized operators required for high-impact actions
- **Ethical prime**: critical misjudgment class (FN/FP/Bias/Integrity)
